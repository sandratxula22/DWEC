Este tutorial presenta los conceptos esenciales del framework Angular 19 presentado el 19 de noviembre de 2024.

Si tiene que trabajar con proyectos legacy (previos a la versión 17) puede tal vez tener la necesidad de ver el tutorial de Angular < 17

Se recomienda para su entendimiento haber realizado los tutoriales de HTML Ya , Css Ya, Javascript Ya.
Luego de completar el curso de Angular esta invitado a aprender JavaScript en el servidor con Node.js..
